# Templates AI

AI-powered templates for support agents. Built with Next.js, Tailwind CSS, and Google Gemini AI.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) to view the app.

## Environment Variables

Create a `.env.local` file with:

```
GEMINI_API_KEY=your_api_key_here
```
